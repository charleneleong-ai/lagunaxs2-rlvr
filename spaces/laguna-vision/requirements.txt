# so we install only its runtime deps — NOT the full package (whose pyproject pulls private deps like
# `autoresearch`, not on PyPI, which breaks the build). compressed-tensors for the NVFP4 base path,
# accelerate for large-model loading, torchvision for the GLM-OCR image processor, tiktoken/
# sentencepiece for the GLM-OCR tokenizer.
#
# torch/torchvision PINNED to cu126: the HF A100 Space driver caps at CUDA 12.9, but the default PyPI
# torch wheel is cu130 -> torch can't see the GPU -> the 66GB base falls back to CPU -> OOM. cu126
# (<=12.9) is the newest build with cp313 wheels for torch 2.12 / torchvision 0.27.
